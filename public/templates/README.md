# Resume Templates

This directory contains company-specific resume templates for optimal ATS compatibility.

## Available Templates:

### Microsoft Resume Template
- **File**: `microsoft-resume.docx`
- **Format**: Professional, clean layout
- **Optimized for**: Microsoft recruitment system
- **Key Features**: Azure skills highlighted, cloud computing focus

### Google Resume Template  
- **File**: `google-resume.docx`
- **Format**: Concise, impact-focused
- **Optimized for**: Google's hiring process
- **Key Features**: Quantified achievements, innovation focus

### Amazon Resume Template
- **File**: `amazon-resume.docx`
- **Format**: Customer-obsessed, metrics-driven
- **Optimized for**: Amazon leadership principles
- **Key Features**: AWS experience, scale achievements

### TCS Resume Template
- **File**: `tcs-resume.docx`
- **Format**: Traditional, academic-focused
- **Optimized for**: TCS recruitment process
- **Key Features**: Academic achievements, project details

### Infosys Resume Template
- **File**: `infosys-resume.docx`
- **Format**: Professional, comprehensive
- **Optimized for**: Infosys hiring criteria
- **Key Features**: Technical skills, certifications

## Usage:
1. Download the template for your target company
2. Customize with your personal information
3. Follow company-specific guidelines
4. Ensure ATS optimization

## Note:
These templates are designed to pass through Applicant Tracking Systems (ATS) and are tailored to each company's specific recruitment criteria and culture.
